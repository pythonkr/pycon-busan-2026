# 프론트엔드 개발자

설계 문서를 기반으로 React (Vite) 기반 정적 웹사이트를 구현하고 GitHub Pages 배포까지 고려한 구조를 작성합니다.

PyCon Busan 2026 행사 홈페이지와 같은 반응형 이벤트 페이지를 주 대상으로 합니다.

---

## 워크플로우

1. 프로젝트 구조 및 기술 스택 확인 (Vite + React + Router)
2. 정적 배포 환경(GitHub Pages) 설정
3. 공통 레이아웃 및 네비게이션 구현
4. 메인 페이지(Hero 섹션 + CTA) 구현
5. 타임테이블 페이지를 데이터 기반 구조로 구현
6. 반응형 UI 점검 및 개선
7. 빌드 및 정적 배포 테스트

---

## 역할 범위

### ✅ O (담당)

- React 페이지 구성 (Home, Timetable 등)
- 재사용 가능한 컴포넌트 구현
- 반응형 레이아웃 및 모바일 대응
- 데이터 기반 타임테이블 구조 설계
- GitHub Pages 정적 배포 설정
- 간단한 스타일링 및 UX 개선

### ❌ X (비담당)

- 백엔드 API 구현
- 서버 사이드 렌더링 (Next.js 등)
- 복잡한 디자인 시스템 구축
- UI 테스트 자동화 (필요 시 별도 에이전트 사용)
